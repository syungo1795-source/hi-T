💪 筋トレ部位当てランダム・マスター
筋トレの種目名から、その種目で主に鍛えられる筋肉（主働筋）を当てる 4 択クイズアプリです。 トレーニング初心者から上級者まで、正しい知識を楽しく身につけることができます。

🚀 アプリ試用 URL
以下のリンクから実際にアプリを体験いただけます： 筋トレ部位当てクイズをプレイする

✨ 主な機能
ランダム出題システム

全 30 種目の豊富なデータから、プレイごとにランダムで 10 問 を選出して出題します。

選択肢のシャッフル機能

正解の選択肢の位置が毎回変わるため、場所で覚えることができず、正確な知識が試されます。

データの永続化（Supabase 連携）

クラウドデータベース Supabase を利用しており、アプリを閉じても過去の回答履歴やみんなのプレイ記録が消えずに保存されます。

リアルタイム進捗表示

プログレスバーにより、全 10 問中の現在位置を視覚的に確認できます。

回答履歴の表示

クイズ終了後、最新のプレイ履歴をデータベースから取得して表示します。

🛠️ 使用技術
Frontend/App Framework: Streamlit

Database: Supabase (PostgreSQL)

Language: Python 3.x

Library: pandas, supabase-py, random

⚙️ セットアップ方法（開発者向け）
このリポジトリをローカル環境や自身の Streamlit Cloud で動かす場合は、以下の設定が必要です。

1. 必要なライブラリのインストール
Bash
pip install -r requirements.txt
2. 環境変数の設定 (Secrets)
Streamlit の Secrets または .streamlit/secrets.toml に、自身の Supabase API 情報を登録してください。

Ini, TOML
SUPABASE_URL = "YOUR_SUPABASE_URL"
SUPABASE_KEY = "YOUR_SUPABASE_ANON_KEY"
3. データベースの準備
Supabase の SQL Editor で以下のテーブルを作成してください。

SQL
create table quiz_logs (
  id bigint generated by default as identity primary key,
  user_name text not null,
  exercise_name text not null,
  is_correct boolean not null,
  created_at timestamp with time zone default now()
);
alter table quiz_logs disable row level security;
